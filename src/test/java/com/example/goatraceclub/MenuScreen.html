VELKOMMEN TIL KLUBBEN FOR GOATS

Om Os

Velkommen til Gedeklubben! Vi er en passioneret forening for alle, der elsker geder og alt, hvad der gør dem unikke. Vores klub samler entusiaster og eksperter, der deler en fælles interesse for de mange forskellige ged racer. Gennem udstillingsshows og arrangementer fejrer vi gederne og fremmer viden om deres avl og pleje. Bliv en del af fællesskabet og oplev glæden ved disse fantastiske dyr!
